Official medical appointment booking system for doctors

== Description ==
This plugin integrates the Emonshi booking system into your WordPress website, allowing patients to book medical appointments online.

Features:
* Easy integration with shortcode
* Real-time availability calendar
* Automatic appointment management
* User-friendly interface

== Installation ==
1. Upload the plugin files to `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode [emonshi_booking] in any page or post

== Frequently Asked Questions ==
= How do I display the booking calendar? =
Simply add the shortcode [emonshi_booking] to any page or post where you want the calendar to appear.
